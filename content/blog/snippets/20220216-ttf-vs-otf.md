## Glossaries

### Bezier Curve

- 베지어 곡선 = N개의 점으로부터 만들어지는 곡선
- 2차 배지어 곡선 (Quadratic Bezier) = 2개의 점 사용
- 3차 배지어 곡선 (Cubic Bezier) = 3개의 점 사용

## TTF

- 1980년대 후반 개발
- TTF = True Type Font
- MS와 Apple이 Adobe에 대항하기 위해 제작
- **Quadratic Bezier** 사용
- OTF보다 빠름
- 모든 응용 소프트웨어에서 사용 가능
- 라이트/볼드/이탤릭 등의 가족체를 하나로 묶을 수 **없음**

## TTF

- 1996년 개발
- OTF = Open Type Font
- MS와 Adobe가 제작
- **Cubic Bezier** 사용
- TTF보다 섬세함
- 모든 응용 소프트웨어에서 사용 가능
- 라이트/볼드/이탤릭 등의 가족체를 하나로 묶을 수 **있음**
